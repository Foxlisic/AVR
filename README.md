# Описание портов

## ЧТЕНИЕ

```
$00     Последний ASCII символ от клавиатуры
$01     Значение 100 Гц таймера
$02     [0] =1 Был получен новый символ. При чтении сбрасывается
$03     [0] =1 Был получен VBlank, конец рисования кадра
$04     [7]     SD Done :: Успешное выполнение команды
        [6]     SD Busy :: 1-Устройство занято
        [5:4]   SD Card Type
        [3:0]   SD Error
$05     Мышь X=0..255
$06     Мышь Y=0..191
$07     [7] =1 Принят сигнал с мыши
        [2] Средняя кнопка мыши
        [1] Правая
        [0] Левая
```

## ЗАПИСЬ

```
$00     [2:0] Цвет бордюра
$01     [0] Номер видеостраницы 0=$8000, 1=$A000
$02     SD LBA [ 7: 0]
$03     SD LBA [15: 8]
$04     SD LBA [23:16]
$05     SD LBA [31:24]
$06     SD CMD =0 Читать =1 Писать
```

# Маппер памяти

```
$0000-$00FF Порты
$0100-$7FFF Общая память
$8000-$9FFF Видеостраница #1
$A000-$BFFF Видеостраница #2
$C000-$FBFF Общая память
$FC00-$FFFF SD i/o 1K
```
