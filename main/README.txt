Порты эмулятора

IO      Описание
------------------------------------------------------------------------
00  RW  Cursor.X
01  RW  Cursor.Y
02  W   PSET (COLOR)
02  R   Microsec/100 :: 100kHz
0C  R   ASCII клавиша (00-7F), отжатая (80-FF)
0D  RW  Цвет бордера [0..15]

        ; Конфигурация видеоадаптера
0E  RW  Бит 0: Видеостраница активная
        Бит 1: Видеостраница отображаемая
        Бит 2: Инкремент по X
        Бит 3: Инкремент по Y

        ; Данные от портов
0F R    Бит 0: Порт PS/2 CLK
        Бит 1: Порт PS/2 DAT
------------------------------------------------------------------------
